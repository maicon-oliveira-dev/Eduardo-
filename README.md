# Blog Jurídico Profissional - Estrutura Base

Projeto base para blog de um advogado especialista, inspirado em grandes blogs jurídicos.
Tecnologias:
- Node.js + Express
- EJS (templates)
- HTML5, CSS3, JavaScript
- Estrutura preparada para painel administrativo simples.

Para iniciar:
1. `npm install`
2. Renomeie `.env.example` para `.env` e ajuste `SESSION_SECRET`.
3. `npm run dev`
4. Acesse `http://localhost:3000`
